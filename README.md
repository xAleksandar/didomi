# Didomi Entry Task

Features React Frontend and NodeJS server as Backend.

# Setup

1. Server Setup
```bash
cd server
npm install
npx ts-node src/server.ts
```

2. Frontend Setup
```bash
cd frontend
npm install
npm start
```

# Tests (Rendering tests & Logical tests incluided)

In order to run frontend tests, run:
```bash
npm test
```
